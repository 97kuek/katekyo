# Architecture Decision Records

ADRは、後からコードだけでは復元できない重要な設計判断を記録する。

## 一覧

| ADR | 状態 | 決定 |
| --- | --- | --- |
| [ADR-0001](0001-documentation-source-of-truth.md) | Accepted | 文書の正本と追跡性を定義する |
| [ADR-0002](0002-google-authentication.md) | Proposed | Googleアカウント認証を安全に段階導入する |

## 状態

- Proposed: レビュー中
- Accepted: 採用中
- Superseded: 新しいADRに置換済み
- Deprecated: 現在は利用しない

採用後のADR本文は書き換えず、判断が変わった場合は新しいADRから置換元を参照する。
